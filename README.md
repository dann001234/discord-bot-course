# Discord Bot com Discord.js e TypeScript — Curso Completo

Curso completo e totalmente autoral para criar bots profissionais para Discord com **Discord.js v14** e **TypeScript**. Do zero absoluto ao nível avançado.

## Como Usar

Este é um site estático. Para abrir o curso:

1. Faça o download dos arquivos
2. Abra o arquivo `index.html` diretamente no navegador, **ou**
3. Hospede os arquivos em qualquer serviço de hospedagem estática (GitHub Pages, Vercel, Netlify, etc.)

## Hospedagem no GitHub Pages

1. Crie um repositório no GitHub
2. Faça upload dos arquivos `index.html`, `css/style.css` e `js/app.js`
3. Vá em Settings > Pages > Source: Deploy from a branch > main
4. Acesse o link gerado pelo GitHub Pages
